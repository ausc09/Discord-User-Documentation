## Introduction

Discord is a free communication platform that allows users to send messages, share files, and host voice and video calls. This guide provides step-by-step instructions for setting up and using Discord for team collaboration in the CST program.

For more information about Discord's full range of features, visit the [Discord website](https://discord.com).

---

## Is This Guide for You?

This guide is written for term 1 CST students who have no prior experience with Discord. The instructions work on any operating system, including macOS and Windows.

---

## By the End of This Guide, You Will Be Able To

1. Download, install, and set up a Discord account
2. Create and configure a Discord server
3. Create channels and manage permissions in a Discord server
4. Host a team meeting using Discord

---

## Prerequisites

Before you begin, make sure you have the following:

- A computer running macOS or Windows
- A valid email address
- A stable internet connection
- Basic familiarity with mouse and keyboard operations
