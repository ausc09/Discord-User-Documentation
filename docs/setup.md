# Getting Started

## Introduction

Discord is a communications platform that enables you to communicate with lecturers and classmates during your time in BCIT. This guide explains how to create an account, add friends and start your first conversation.

---

## Prerequisites

- You have an email account.
- You have installed the Discord application.

---

## Create An Account

To use Discord, you must first create an account. This section will show you how to register a free Discord account.

1. **Click** [Register] to create a new account.
   ![Launch](image/setup/launch.png)

2. **Enter** your email address, username, password, date of birth and **click** [Create Account] button.
   ![Create Account](image/setup/create.png)

3. Your account is created, and you will be directed to the main page. 
   ![Home Page](image/setup/main.png)

!!! note
    You can verify your email now or later by clicking the link sent to your email.

!!! success
    Your Discord account has been successfully created.

---

## Add Friend

After creating an account, you can add your classmates to start a conversation with them. This section will show you how to add friend in Discord.

### Send Request

1. **Click** [Friends] button direct to friends manage page.
   ![Friends](image/setup/add.png)

2. **Click** [Add Friend] button.
   ![Friends Page](image/setup/friend.png)

3. **Enter** the username of your friends and **click** [Send Friend Request] button.
   ![Add Friends](image/setup/username.png)

4. **Click** [Pending] button to view all the pending request you send.
   ![Pending Request](image/setup/pending.png)

5. **Click** [All] button to view list of your friend.
   ![Friend List](image/setup/list.png)

!!! success
    Your friend request has been successfully sent.

### Receive Request

1. **Click** [Friends] button direct to friends manage page.
   ![Friends](image/setup/add.png)

2. **Click** [Pending] button to view all the pending request.
   ![Friends Page](image/setup/receive.png)

3. **Click** [Accept] button if you want to accept request.
   ![Accept](image/setup/clickaccept.png)

4. **Click** [Accept] button again to confirm friend request.
   ![Confirm](image/setup/confirm.png)

5. Friend added and direct to the friend list.
   ![Confirm](image/setup/friendlist.png)

!!! success
    You have successfully accepted the friend request.

---

## Start Chat

After adding a friend, you can start your first conversation with them. This section will show you how to start chatting with your friend.

1. **Click** "All" button and choose the friend you want to communicate with and click it.
   ![Click Friend](image/setup/start.png) ![Chat](image/setup/empty.png)

2. **Enter** your message and send it.
   ![Chat](image/setup/chat.png)

!!! success
    You have successfully started your first chat on Discord.

---

## Conclusion

In this section, you have learned how to:

- Create a Discord account  
- Add friends  
- Send and receive friend requests  
- Start a conversation  

You are now ready to use Discord for communication.
