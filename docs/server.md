# Create and Configure a Discord Server

## Introduction

This section explains how to create and configure a Discord server for a **term project team (3–10 members)**.

For more information about Discord features, visit the [Discord website](https://discord.com).

---
## Prerequisites

- You have a Discord account and are signed in.
- You are using Discord desktop app or web version.
- Your team size is within 3–10 members.

!!! note
    This setup is optimized for term project collaboration, not public community servers.

---
## Recommended Configuration (Term Project Scope)

- **Default Notifications:** Only @mentions  
- **Invite Expiration:** 7 days  
- **Invite Max Uses:** 10–20  
- **Safety Filters:** Enabled

---
## Setup Steps

1. Open Discord and click **+ (Add a Server)**.  
![Add a Server](image/server_setup/t2_s1_add_server.png)

2. Select **Create My Own** and choose the server type.  
   **Why:** This creates a custom server suited to your team’s use case.
![Choose the server type](image/server_setup/t2_s1_create_your_own.png)

3. Choose the server purpose, if not sure. can skip this question.
![Choose the server purpose](image/server_setup/t2_s1_choose_server.png)


4. Enter a clear server name and optionally add icon, then click **Create**.  
   **Why:** A clear name helps teammates identify the correct server quickly.
   ![Enter name](image/server_setup/t2_s1_create_server.png)

5. Open the server dropdown and click **Server Settings**.  
   **Why:** All core configuration options are managed in Server Settings.
   ![open dropdown](image/server_setup/t2_s5%20a.png)
   You will see the dropdown menu like this.
   ![click](image/server_setup/t2_s5%20b.png)


6. In **Server Settings**, scroll all the way down the left sidebar and click **Enable Community**, then click **Enable Community** on the right side to complete setup.
   **Why:** Community setup enables useful management and baseline safety features.
   ![click](image/server_setup/t2_s6_enable%20community.png)


7. In the **Community Setup**, complete **Safety Checks**, check **Scan media content from all members**, then click **Next**.
 **Why:** These filters reduce inappropriate content and keep collaboration space clean.
   ![click](image/server_setup/t2_s7_safetycheck.png)

8. In **Community Setup**, Discord may auto-skip **Setting Up the Basics** by applying defaults automatically (for example, **Default Notifications: Mentions Only** and **Remove risky permissions from @everyone**). When it reaches **Finish Touches**, check **I agree and understand**, then click **Finish Setup**.
**Why:** This reduces notification overload while preserving important alerts.
 ![click](image/server_setup/t2_s8_finish_setup.png)

9. After Community setup is completed, you may hear a Discord notification sound. This is normal, we will review notifications later. Now click **Invites** in the left sidebar.
![click](image/server_setup/t2_s9_invite.png)

10. In the **Invites** section, find **Create Invite Link** on the right side and click it.
![click](image/server_setup/t2_s10_create_link.png)

11. In the pop-up window, choose one of the following:
      - Click **Invite** to let Discord send an invite directly.
      - Click **Copy** to copy the invite link and send it to your teammate manually.
   ![click](image/server_setup/t2_s11_invite_friend.png)

!!! success
    Core server setup is complete and ready for your next tasks.

---
## Summary

Your Discord server is now configured for term project collaboration with appropriate identity, safety, notification, and invite settings for a 3–10 member team.
