# Glossary

| Term | Definition |
|------|------------|
| **Channel** | A dedicated space within a server for text, voice, or video communication. |
| **Community Server** | A server type with additional safety and moderation features enabled. |
| **Friend Request** | An invitation sent to another user to connect on Discord. |
| **Invite Link** | A shareable URL that allows others to join a specific server. |
| **Permission** | A setting controlling what actions a user can perform in a server or channel. |
| **Role** | A label assigned to members that groups them by access level or function. |
| **Server** | A shared workspace containing channels, members, and settings. |
| **Username** | A unique identifier for each Discord account used to find and add friends. |
| **Voice Channel** | A channel for real-time audio, video, and screen sharing communication. |
| **@mention** | A way to notify a specific user by typing @ followed by their username. |
